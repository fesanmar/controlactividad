==== Response status

|===
|Status|Descripción

|200
|Todo correcto, retorna la información requerida

|404
|No se ha encontrado el curso cuyo código se especificó. Respuesta vacía.
|===

==== Response types

|===
|Tipo|Descripción

|application/json
|Array conteniendo objetos como los mostrado abajo

|application/xml
|Árbol conteniendo objetos. TODO: generar salida xml

|application/html
|Tabla HTML con un campo por columna y una fila por recurso. TODO: generar salida HTML.

|image/png, image/jpg
|Imagen representando un gráfico de barras como el mostrado abajo

|text/csv
|Fichero de texto separado por comas. TODO: implementar y documentar
|===
